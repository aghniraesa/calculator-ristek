# Hello App

## Description
Aplikasi kalkulator sederhana yang dapat menjumlahkan, mengurangi, mengalikan, 
dan membagi dua angka.

## Requirements
- Gradle 5.6.4
- Java 8 or greater

## Installation Guide
1. Clone this repo
2. 
```
./gradle build
```

## Run Guide
```
./gradle run
```
